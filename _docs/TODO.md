### 代办组件

- 气泡组件(类似直播间的左下角的气泡)
- 契约组件扩展
  - `ui.search` 搜索下拉组件, 模糊搜索名称(指定下拉条数、指定字段名称(keyword)、指定默认参数)
  - `ui.random` 随机编号组件, 指定编号前缀、长度、是否默认生成。

| 属性          | 描述         |
| ------------- | ------------ |
| defaultParams | 默认请求参数 |
| useAction     | 请求处理组   |
| initialValues | 初始化值     |
| xx$           | 事件         |
